Clock 1: Shy Clock

This clock only catches up to real time when no one is looking at it. The fewer connections there are, the more reliable the time.

Clock 2: Voyeur Clock

This clock catches up to real time faster when there are more people observing it. The more connections there are the more reliable the time.

Clock 3: Universal Clock

The hour hand of this clock cycles once for the predicted lifespan of the universe (assuming heath death trajectory).
The minute hand cycles once for the lifespan of planet earth (assuming solar expansion).
The second hand cycles once for the lifespan of the human race (assuming the Doomsday 2019 predictions).

Clock 4: Richard Simmons Clock

This clock works best when you are moving. Currently confined to screen, with plans to use movement data from smart devices.

Clock 5: Binary Timer

This timer counts up in binary

Clock 6: Astrolabe Clock

This device can tell the time on any planet anywhere, as long as you have a star map and know some info about the planet.
